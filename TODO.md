# Supervillain TODO

Issue tracking lives in **kata** (project `supervillain`), bound to this
workspace via `.kata.toml`.

- `kata list` — open issues
- `kata ready` — open issues with nothing blocking them
- `kata show <ref>` — full issue + comments
- `kata tui` — interactive browser

Everything previously tracked in this file, plus the SuperVillain Linear
project (THE-xxx issues), was migrated into kata on 2026-07-07. Each migrated
issue carries a provenance footer linking back to its Linear original. For
shipped work, see [CHANGELOG.md](CHANGELOG.md).
